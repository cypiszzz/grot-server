GROT
====

Requirements
------------

* python3
* [tornadoweb](http://www.tornadoweb.org/)
* [toro](http://toro.readthedocs.org/)

Install
-------

	mkvirtualenv grot-server -p /usr/bin/python3.4
	pip3 install -r requirements.txt

Run
---

### Server

	workon grot-server
	python3 server.py


Client
------

For details check
