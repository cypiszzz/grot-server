GROT
====

Requirements
------------

* python3
* [tornadoweb](http://www.tornadoweb.org/)
* [toro](http://toro.readthedocs.org/)

Install
-------

	pip3 install -r requirements.txt

Run
---

### Server

	python3 server.py

### Client

	python3 client.py [Name] [Game]

Games
-----

### Contest

To run the Contest game click on 'GROT - Score Board'
